## Installation
* Download latest release
* Add it to your resource folder
* Edit your **server.cfg**
```
start vRP_Repair
```
## How to use
Item Name is **fixtool**. Click **Toolbox**, You can see **Use** Menu. If you press this Menu near the vehicle, repair the vehicle.
